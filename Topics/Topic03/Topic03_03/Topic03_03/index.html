    Topic 3.3 - R 中图像的修饰¶

    本节我们来介绍一下在 R 中如何对图像进行修饰，首先我们导入包和数据：

    library(tidyverse)  # for plotting, includes ggplot
library(patchwork)  # for combining multiple plots into subfigures
library(scales)     # for formatting axis scales
library(ggokabeito) # color blind friendly color palette -- this course's default

beer <- read_csv("beer.csv")

    1. ggplot2 中颜色的使用¶

    在 ggplot2 中，有两个控制颜色的参数：

    • color：控制图形元素的边框颜色
    • fill：控制图形元素的填充颜色

    注意，这两个参数是既可以放到 aes 函数中，也可以放到 geom 系列函数中：

    • 放在 aes 函数中通常填入一个列名，例如 color = imported，表示按照这个列来区分不同颜色的图形元素，因为 aes 函数本身的含义就是数据如何延伸如何区分
    • 放在 geom 系列函数中通常填入一个具体的颜色值，例如 color = "red"，表示所有的图形元素都使用这个颜色

    我们来在箱形图中演示一下这两个参数的使用：

    • 首先我们在 aes 函数中使用 fill = imported 来指定按照进口和非进口啤酒来区分不同颜色的箱体：
    ggplot(beer) +
  geom_boxplot(aes(y = price, 
                   x = imported, 
                   fill = imported))

    • 这里，我们在 aes 函数中，既使用了 x = imported 来指定进口和非进口啤酒箱体按 x 轴排列，又使用了 fill = imported 来指定按照进口和非进口啤酒来区分不同颜色的箱体了，也就是说，进口与非进口这件事我们既用位置来区分，也用颜色来区分

    接下来，我们尝试着，把 color 参数放到 geom_boxplot 函数中

    • 我们在 geom_boxplot 中设置 fill = "lightblue"，表示所有的箱体都使用浅蓝色的填充颜色：
    ggplot(beer) +
  geom_boxplot(aes(y = price, 
                   x = imported), 
                fill = "lightblue")

    • 这里所有的箱体就都用淡蓝色来填充了，并不会按照什么要素去区分

    我们本门课在图像绘制时，基本上都使用 Okabe–Ito palette 这个调色板，这个是你们课上推荐的：

    • 要想使用这个调色板，可以在 ggplot 代码块中加一段 scale_fill_okabe_ito() 就可以了，这样就会自动使用 Okabe–Ito palette 这个调色板来区分不同的颜色了
    • 例如，我们在箱形图中使用 scale_fill_okabe_ito() 来指定使用 Okabe–Ito palette 这个调色板来区分进口和非进口啤酒的颜色：
    ggplot(beer) +
    geom_boxplot(aes(y = price, 
                     x = imported, 
                     fill = imported)) +
    scale_fill_okabe_ito()

    除了颜色之外，我们还可以设置透明度：

    • 透明度的参数是 alpha
    • 它的取值范围是 0 到 1，0 表示完全透明，1 表示完全不透明，默认值是 1

    我们来看在箱形图中设置 alpha 参数的例子：

    ggplot(beer) +
    geom_boxplot(aes(y = price, 
                     x = imported, 
                     fill = imported),
                 alpha = 0.35) +
    scale_fill_okabe_ito()

    2. ggplot2 中设置主题¶

    主题（Theme）是 ggplot2 中一个非常重要的概念：

    • 它控制了图像的整体外观和风格，包括背景、网格线、字体、图例等元素的样式
    • 通过设置主题，我们可以让图像看起来更美观、更专业，或者更符合特定的风格要求

    在 ggplot2 中，我们有两种方式设置主题：

    • 一种是使用预设的主题，例如：

      • + theme_gray()：这是 ggplot2 的默认主题，具有灰色背景和白色网格线
      • + theme_bw()：这是一个黑白主题，具有白色背景和黑色网格线
      • + theme_minimal()：这是一个极简主题，具有白色背景和最少的网格线
      • + theme_classic()：这是一个经典主题，具有白色背景和黑色网格线，但没有坐标轴线

    • 还有一种是使用 theme() 函数来自定义主题，其中要自己设置各个元素的样式，例如：

      • + theme(panel.background = element_rect(fill = "lightgray"))：设置面板背景为浅灰色
      • + theme(panel.grid.major = element_line(color = "white"))：设置主网格线为白色
      • + theme(panel.grid.minor = element_blank())：去掉次网格线
      • + theme(axis.text = element_text(size = 12, color = "blue"))：设置坐标轴文本的大小和颜色
      • + theme(legend.position = "top")：将图例放在顶部

    • 注意，这两种设置主题的方式并不冲突：

      • 我们可以先使用预设主题来快速设置一个基础的图像风格
      • 然后再使用 theme() 函数进行微调，例如修改字体、图例位置、背景颜色等细节，以达到我们想要的效果

    首先我们来看一下，ggplot2 中预设的主题的区别：

    p <- ggplot(beer) +
    geom_boxplot(aes(y = price, x = imported, fill = imported)) +
    scale_fill_okabe_ito()
(
    (p + theme_grey()   + labs(title = "theme_grey()"))   |
    (p + theme_bw()     + labs(title = "theme_bw()"))  
    )   /
( 
    (p + theme_classic() + labs(title = "theme_classic()")) |
    (p + theme_minimal() + labs(title = "theme_minimal()"))
)

    接着，我们来看一个预设主题与自定义主题的结合使用的例子：

    ggplot(beer) +
geom_boxplot(aes(y = price, x = imported, fill = imported)) +
scale_fill_okabe_ito() + 
theme_minimal() +
theme(legend.position = "none")

    3. ggplot2 中的文本元素¶

    在 ggplot2 中，我们可以使用 labs() 函数来设置图像的标题、副标题、坐标轴标签等文本元素：

    • title：设置图像的主标题
    • subtitle：设置图像的副标题
    • caption：设置图像的说明文字，通常放在图像的右下角
    • x：设置 x 轴的标签
    • y：设置 y 轴的标签
    labs(
    title = "Main Title", 
    subtitle = "Subtitle", 
    caption = "Caption",
    x = "X-axis Label", 
    y = "Y-axis Label"
)

    例如，我们在箱形图中使用 labs() 来设置标题，副标题和坐标轴标签：

    ggplot(beer) +
geom_boxplot(aes(y = price, 
                x = imported, 
                fill = imported)) +
scale_fill_okabe_ito() + 
labs(
    title = "Weekly Beer Prices by Import Status",
    subtitle = "Dominicks Finer Foods Stores",
    caption = "Source: University of Chicago Booth School of Business Kilt's Center for Marketing Research",
    x = "Production Location",
    y = "Price (USD)"
) +
theme_minimal() +
theme(legend.position = "none")

    在 ggplot2 中，我们还支持修改文本元素的样式，例如字体、大小、颜色等，这些设置通常也是通过 theme() 函数来实现的

    • 事实上，lab 函数只是用来设置文本元素的内容的
    • 而 theme 函数才是用来设置文本元素的样式的

    在 theme() 函数中，我们可以使用 element_text() 来设置文本元素的样式，例如：

    theme(
    plot.title = element_text(size = 16, face = "bold", color = "red", hjust = 0.5),
    plot.subtitle = element_text(size = 14, face = "italic", color = "blue"),
    plot.caption = element_text(size = 10, face = "plain", color = "gray"),
    axis.title.x = element_text(size = 12, face = "bold", color = "green", angle = 45, hjust = 1),
    axis.title.y = element_text(size = 12, face = "bold", color = "green")
)

    • 首先，theme 函数中的参数分别表示：

      • plot.title：设置主标题的样式
      • plot.subtitle：设置副标题的样式
      • plot.caption：设置说明文字的样式
      • axis.title.x：设置 x 轴标签的样式
      • axis.title.y：设置 y 轴标签的样式

    • 其次，element_text 函数中的参数分别表示：

      • size：设置文本的大小
      • face：设置文本的字体样式，例如 "plain"（正常）、"bold"（加粗）、"italic"（斜体）等
      • color：设置文本的颜色
      • hjust：设置水平对齐方式，0 表示左对齐，0.5 表示居中，1 表示右对齐
      • angle：设置文本的旋转角度，例如 45 表示旋转 45 度

    我们来看一下在箱形图中设置文本元素样式的例子：

    ggplot(beer) +
geom_boxplot(aes(y = price, x = imported, fill = imported)) +
scale_fill_okabe_ito() + 
labs(
    title = "Weekly Beer Prices by Import Status",
    x = "Production Location",
    y = "Price (USD)"
) +
theme_minimal() +
theme(
    legend.position = "none",
    axis.text.x = element_text(angle = 45, hjust = 1),
    plot.title = element_text(hjust = 0.5) 
)

    4. ggplot2 中的坐标轴修饰¶

    在 ggplot2 中，我们可以使用 scale_y_continuous() 和 scale_x_continuous() 函数来修饰坐标轴：

    • 事实上，scale_y_*() 与 scale_x_*() 是一个系列的函数
    • 这里的 * 不仅可以是 continuous，还可以是 discrete、date、datetime 等等，分别对应不同类型的坐标轴
    • 我们这里先来看坐标轴数据是连续型的情况

    在 scale_y_continuous() 和 scale_x_continuous() 函数中，我们可以使用以下参数来修饰坐标轴：

    • breaks：设置坐标轴刻度的位置，可以是一个数值向量，表示刻度的具体位置
    • labels：设置坐标轴刻度的标签，可以是一个字符向量，表示每个刻度对应的标签
    • limits：设置坐标轴的范围，可以是一个长度为 2 的数值向量，表示坐标轴的最小值和最大值
    • expand：设置坐标轴的扩展范围，可以是一个长度为 2 的数值向量，表示坐标轴在数据范围之外的扩展比例

    我们来看一下在箱形图中使用 scale_y_continuous() 来修饰 y 轴的例子：

    ggplot(beer) +
geom_boxplot(aes(y = price, x = imported, fill = imported)) +
scale_y_continuous(
    breaks = c(3, 3.5, 4, 4.5, 5, 5.5, 6),
    labels = scales::dollar
) +
scale_fill_okabe_ito() +
theme(legend.position = "none") +
labs(y = "Price", x = "Production Location") +
theme_minimal()
