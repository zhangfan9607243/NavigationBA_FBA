    Topic 3.4 - 完整的图像构建展示¶

    首先，我们先导入必要的包和数据：

    library(tidyverse)  # for plotting, includes ggplot
library(patchwork)  # for combining multiple plots into subfigures
library(scales)     # for formatting axis scales
library(ggokabeito) # color blind friendly color palette -- this course's default

beer <- read_csv("beer.csv")

    1. 单个图像的构建展示¶

    本节我们来将之前学习的代码构建完整的图像构建展示一下：

    • 首先，我们构建一个散点图，来展示啤酒的价格和数量之间的关系

    • 我们先构建一些简单的图像要素：

      • 使用颜色区分进口和非进口啤酒
      • 使用透明度来减少点的重叠
      • 使用 scale_color_okabe_ito 来设置颜色方案，使得图像更加美观和易于区分
    ggplot(beer) +
geom_point(
    aes(y = price, x = qty, color = imported),
    alpha = 0.25
) +
scale_color_okabe_ito()

    在这一版本的图像中，我们发现 X 轴 和 Y 轴的范围比较大：

    • 导致大部分点堆在左下角，一些价格和数量的极端值导致图像的范围过大了
    • 这时候，我们使用 scale_x_continuous 和 scale_y_continuous 来设置 x 轴和 y 轴的范围，来让图像更加聚焦在我们感兴趣的范围内了
    • 在这两个函数中，我们使用 trans = "log" 来设置 x 轴和 y 轴的刻度为对数刻度，这样就可以更好地展示数据的分布
    • 我们再加上一些额外修饰，例如修改 x 轴和 y 轴的标签，以及在 y 轴上添加美元符号来表示价格的单位，这样就可以让图像更加清晰和易于理解了
    ggplot(beer) +
geom_point(
    aes(y = price, x = qty, color = imported),
    alpha = 0.25
) +
scale_color_okabe_ito() +
scale_x_continuous(
  trans = "log",
  breaks = c(1, 2, 5, 10, 20, 50, 100, 200)
) + 
scale_y_continuous(
  trans = "log",
  breaks = c(3, 4, 5, 6),
  labels = scales::dollar
) +
labs(x = "Quantity", y = "Price")

    接着，我们想把进口啤酒和非进口啤酒分成两个面板来展示：

    ggplot(beer) +
geom_point(
    aes(y = price, x = qty, color = imported), 
    alpha = 0.25
) +
facet_wrap(~ imported) +
scale_x_continuous(
    trans = "log",
    breaks = c(1, 2, 5, 10, 20, 50, 100, 200)
) + 
scale_y_continuous(
    trans = "log",
    breaks = c(3, 4, 5, 6),
    labels = scales::dollar
) +
labs(x = "Quantity", y = "Price") + 
scale_color_okabe_ito() +
theme_minimal() +
theme(legend.position = "none")

    之后，我我们想加入平滑曲线，来展示价格和数量之间的关系：

    • 由于散点图和平滑曲线都是价格在 y 轴，数量在 x 轴
    • 所以我们可以直接把 aes(y = price, x = qty) 放在 ggplot() 中，这样就可以让散点图和平滑曲线共享同一个坐标轴了
    ggplot(
  beer, 
  aes(y = price, x = qty)
) +
  geom_point(
    aes(color = imported), 
    alpha = 0.25
) +
geom_smooth(
    method = "lm", 
    color = palette_okabe_ito()[7],
    se = TRUE
) +
facet_wrap(~ imported) +
scale_x_continuous(
    trans = "log",
    breaks = c(1, 2, 5, 10, 20, 50, 100, 200)
) + 
scale_y_continuous(
    trans = "log",
    breaks = c(3, 4, 5, 6),
    labels = scales::dollar
) +
labs(x = "Quantity", y = "Price") + 
scale_color_okabe_ito() +
theme_minimal() +
theme(legend.position = "none")

    [1m[22m`geom_smooth()` using formula = 'y ~ x'

    到此为止，我们就完成了一个完整的图像构建展示了：

    • 这个图像展示了啤酒的价格和数量之间的关系，并且区分了进口和非进口啤酒
    • 通过使用对数刻度，我们可以更好地展示数据的分布，并且通过添加平滑曲线，我们可以更清晰地看到价格和数量之间的关系
    • 最后，通过一些额外的修饰，我们可以让图像更加美观和易于理解了

    2. 多个图像的构建展示¶

    在多图构建的时候，我们常用的方式是使用 | 和 / 来构建多个图像：

    • 但是由于一个图像的代码比较长，所以我们可以分别把每个图像结果赋值给一个变量
    • 然后再使用 | 和 / 来组合这些图像变量，这样就可以让代码更加清晰和易于维护了

    例如，我们将散点图、箱形图、密度图分别赋值给 plt_price_qty、plt_price、plt_quantity 这三个变量：

    plt_price <- 
    ggplot(beer) +
    geom_boxplot(aes(y = price, x = imported, fill = imported)) +
    scale_y_continuous(
        breaks = c(3, 3.5, 4, 4.5, 5, 5.5, 6),
        labels = scales::dollar
      ) +
    scale_fill_okabe_ito() +
    theme(legend.position = "none") +
    labs(y = "Price", x = "Production Location") +
    theme_minimal()+
    theme(legend.position = "none")

    plt_quantity <-
    ggplot(beer) +
    geom_density(
        aes(x = qty, fill = imported), 
        alpha = 0.35
    ) +
    scale_x_continuous(
        trans = "log",
        breaks = c(1, 2, 5, 10, 20, 50, 100, 200)
    ) +
    labs(y = "Density", x = "Quantity Sold") +
    scale_fill_okabe_ito() +
    theme_minimal() 

    plt_price_qty <- 
    ggplot(
        beer, 
        aes(y = price, x = qty)
    ) +
    geom_point(
        aes(color = imported), 
        alpha = 0.25
    ) +
    geom_smooth(
        method = "lm", 
        color = palette_okabe_ito()[7],
        se = TRUE
    ) +
    facet_wrap(~ imported) +
  scale_x_continuous(
    trans = "log",
    breaks = c(1, 2, 5, 10, 20, 50, 100, 200)
  ) + 
  scale_y_continuous(
    trans = "log",
    breaks = c(3, 4, 5, 6),
    labels = scales::dollar
  ) +
  labs(x = "Quantity", y = "Price") + 
  scale_color_okabe_ito() +
  theme_minimal() +
  theme(legend.position = "none")

    之后，我们使用 plt_price_qty / (plt_price | plt_quantity) 来把这三个图像组合成一个多图展示了：

    plt_price_qty /
(plt_price | plt_quantity)

    [1m[22m`geom_smooth()` using formula = 'y ~ x'

    这样，我们就构建完成了一个包含散点图、箱形图和密度图的多图展示了
